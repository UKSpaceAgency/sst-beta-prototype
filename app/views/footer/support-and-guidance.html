Using the service

Monitor space hazards aims to provide clear, actionable data to help its users make informed decisions regarding potential collisions. In order to do this, it is important to understand how to use the service.

We have provided a brief video [5 minutes] that will introduce you to the main features of Monitor space hazards, including:

  • An overview of the main pages on the site
  • How to set your own event notification thresholds settings to determine what events are of high interest to you
  • How to choose your notification settings

There is also additional material provided on our technical documentation site, on:

  • How to upload and manage ephemeris data
  • How to use the API

Adding users to your organisation

The organisation administrator is able to:

  • View organisation details
  • Add a new user or remove existing users
  • Amend a user's details

This can be done from the account page on the site.

The support process

Monitor space hazards is expected to run reliably at all times and available 24/7. However, like all web services, bugs can be introduced and unexpected downtime can occur, at which point support will be necessary.

Technical support for Monitor space hazards is available 9-5, Monday to Friday. In the event of an issue occuring outside of Monday to Friday 9-5 we will respond the next working day.

If you have any questions, or need any support whilst using the service, email monitorspacehazards@ukspaceagency.gov.uk with any queries.
