Help with this table

Custom thresholds set means you will receive notifications when a conjunction event involving any satellite selected for that group meets all selected thresholds.

Notifications disabled means we will not send any notifications about conjunction events for satellites in this group.

Do not notify for conjunctions with lets you stop notifications when satellites in this group are in a conjunction with a specific object. This can be useful when you are running tests with that object.

You can change or delete each of the custom settings listed in the table by selecting 'edit' or 'delete'.
