Event data

UKSA orbital analysts carry out conjunction event analysis.

UKSA orbital analysts use several data sources to perform their conjunction analysis.

These include:

  • Satellite operator-uploaded ephemeris data
  • SpaceTrack SATCAT and CDMs
  • ESA-DISCOS (European Space Agency's Database and Information System Characterising Objects in Space)

All other event data is from Space-Track.
